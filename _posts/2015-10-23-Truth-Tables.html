Basic Laws

A’ or ¬A = NOT A

AuB or A+B = A OR B

AnB or A*B = A AND B

(A’)’ = A

A*A’ = 0

A+A’ = 1

Order of operations - NOTs, ANDs, ORs.

DeMorgans Laws

Change all ANDs to ORs or ORs to ANDs, NOT all values. NOT NOT A = A
e.g. (A*B)’ = (A’+B’)

Karnaugh Maps
Examples
  1. (A*A’) + B = 0 + B = B
  2. (A+B) * (A*C) = A + B
  3. (A*B)’ + (A’*B) = A’+B + A’*B = A’(1+B) + B = A’+B
  4. AB + (A’B) = B(A + A’) = B(0) = B
  5. (A*B’)’ = A’+B
  6. ((A’*B)*(B*C’)*(A’*C’))’ = A+B’ + B’+C + A+C = A + B’ + C
